## Sistema de Biblioteca em Python:

Projeto em Python que simula um sistema de biblioteca simples.

##  Funcionalidades:
- Cadastro de livros  
- Cadastro de usuários  
- Empréstimo de livros  
- Devolução de livros  
- Consulta de livros emprestados  

## Objetivo:
Praticar lógica de programação, listas, dicionários e estrutura de menus em Python.

## linguagem: 
- Python
